# Atividade 10
Criar dois programas que utilizem Socket TCP em Java. Sendo:

Programa Servidor

Deverá aceitar conexões de diversos clientes;
O servidor irá receber do cliente a marca de um carro e retornará um texto com a explicação sobre o nome da marca;
Se a marca do carro não existir, então deverá retornar uma mensagem informativa.
O servidor deve atender vários clientes ao mesmo tempo.
Programa Cliente

Deve conectar ao servidor (IP 127.0.0.1 e porta 12345)
Deve enviar o nome de uma marca de carro para o servidor e receber de volta uma descrição da marca
Quando informar a palavra sair, deve fechar a conexão com o servidor e sair.
